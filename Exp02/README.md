## Experiment 2
To implement lex programs to identify keywords,uppercase words,vowels in a word and vowel count in a word

### Commands used
* ```shell
  lex <pgm_name>.l
  ```
* ```shell
  yacc -d <pgm_name>.y
    ```
* ```shell
  gcc lex.yy.c y.tab.c -o parser -ll
    ```
* ```
  ./parser
  ```
